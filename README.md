# Netflix Data Cleaning & Reporting Automation

## Project Overview

This project demonstrates an end-to-end data analytics workflow using the Netflix Titles dataset.

The objective is to automate data cleaning, preprocessing, reporting, and dashboard creation using Python and Power BI.

---

## Objectives

- Clean and preprocess raw data
- Handle missing values
- Remove duplicate records
- Generate automated reports
- Create visual summaries
- Build an interactive Power BI dashboard

---

## Dataset

Netflix Titles Dataset

Columns include:

- show_id
- type
- title
- director
- cast
- country
- date_added
- release_year
- rating
- duration
- listed_in
- description
